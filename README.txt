# Pair Code Session ID Generator

## How to Use

1. Install dependencies:

   ```
   npm install
   ```

2. Start the server:

   ```
   npm start
   ```

3. Open the URL shown in the terminal (e.g., http://localhost:3000).
4. Scan the Pair Code with your WhatsApp.
5. Once connected, copy the Session ID displayed.
6. Paste it in your bot's `.env` file like this:

   ```
   SESSION_ID=your_generated_session_here
   ```

## Deploy to Heroku

1. Create a Heroku app.
2. Push this project using Git or Heroku CLI.
3. Open the app and follow the same steps.
